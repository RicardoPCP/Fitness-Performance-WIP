# <img src="https://github.com/RicardoPCP/Fitness-Performance/blob/main/assets/logo.png" alt="logo" width="40" height="auto"/> ACADEMIA FITNESS PERFORMANCE

## Sobre:

Este repositório contém o código-fonte do meu primeiro projeto freelancer focado em desenvolvimento front-end.

O projeto consiste em um website desenvolvido para a academia Fitness Performance, com o objetivo de apresentar seus serviços e canais de contato de forma moderna, responsiva e acessível. O site também oferece funcionalidades interativas para melhorar a experiência dos alunos, como recomendação de músicas para a playlist da academia.

Link de Acesso: https://ricardopcp.github.io/Fitness-Performance/
